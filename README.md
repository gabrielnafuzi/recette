## Atividade Avaliativa da matéria: DESENVOLVIMENTO DE APLICAÇÕES WEB I

## ⚡ Tecnologias Utilizadas

Para o desenvolvimento do projeto foram utilizadas as seguintes tecnologias:

### Front-end
- [Vue 3](https://v3.vuejs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Axios](https://github.com/axios/axios)
- [Pinia (State management)](https://pinia.esm.dev/)
- [Windi CSS](https://windicss.org/)
- [Vue Draggable](https://github.com/SortableJS/vue.draggable.next)
- [Vite plugins](https://vitejs.dev/plugins):
  - [vite-plugin-components](https://github.com/antfu/vite-plugin-components)
  - [vite-plugin-fonts](https://github.com/stafyniaksacha/vite-plugin-fonts)
  - [vite-plugin-pages](https://github.com/hannoeru/vite-plugin-pages)
  - [vite-plugin-purge-icons](https://github.com/antfu/purge-icons)
  - [vite-plugin-vue-layouts](https://github.com/JohnCampionJr/vite-plugin-vue-layouts)
  - [vite-plugin-windicss](https://github.com/windicss/vite-plugin-windicss)

### Back-end
- [Node.js](https://nodejs.org)
- [Express](https://expressjs.com/)
- [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken)
- [bcrypt](https://github.com/kelektiv/node.bcrypt.js)
- [Multer](https://github.com/expressjs/multer)
- [PostgreSQL](https://www.postgresql.org/)
- [Sequelize](https://sequelize.org/)
